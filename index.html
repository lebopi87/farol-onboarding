Como usar
⚙️ Configurar Supabase (banco na nuvem)
  1. Acesse supabase.com e crie uma conta gratuita
  2. Crie um novo projeto (escolha qualquer nome e região)
  3. No painel do projeto, vá em Settings → API
  4. Copie a Project URL e a anon public key
  5. Abra este arquivo HTML num editor de texto (Bloco de Notas ou VS Code)
  6. Encontre as linhas SUPABASE_URL e SUPABASE_KEY e substitua pelos valores copiados
  7. No Supabase, vá em Table Editor → New Table, crie uma tabela chamada onboarding com apenas uma coluna extra: data do tipo text
  8. Em Authentication → Policies, habilite acesso público (anon) para leitura e escrita na tabela
  9. Salve o arquivo HTML e hospede no GitHub Pages
📋 Fluxo de onboarding
  1. Cadastre o profissional e selecione o perfil correto
  2. Em Contratos, avance cada documento: Pendente → Enviado ao Certisign → Assinado
  3. Para Assessores Comerciais, após aprovação na prova, clique em Desbloquear para liberar Ancord e XP
